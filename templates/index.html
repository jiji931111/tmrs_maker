💡 고급 변수 및 포맷 사용 가이드

{% raw %}

1. 기본 변수 텍스트 매핑

%(변수명)s 형태로 사용합니다. (예: %(tmrs_name)s, %(code)s)

2. 글자 수 고정 맞춤 (정렬 패딩)

글자 길이가 다를 때 텍스트 표를 예쁘게 맞추려면 칸을 고정하세요.

  • 좌측 정렬(30칸): %(tmrs_name)-30s
  • 우측 정렬(30칸): %(tmrs_name)30s
  • Jinja2 사용 시: {{ tmrs_name.ljust(30, '_') }} (빈 공간 언더바 처리)

3. 수식 연산 및 함수 적용 (Jinja2)

{{ }} 괄호 안에서는 자유로운 파이썬 연산이 가능합니다.

  • 16진수 → 10진수 덧셈: {{ hex2dec(ori1) + 100 }}
  • 수식 결합: {{ hex2dec(ori1) * 2 + hex2dec(ori2) }}
  • 조건부 텍스트: {{ 'PASS' if hex2dec(code) > 100 else 'FAIL' }}
실전 예시 (복사해서 테스트해보세요):

NAME: [%(tmrs_name)-20s] | CODE: %(code)s | VAL: {{ hex2dec(ori1) + 150 }}
